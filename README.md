# Agentic AI Learning Lab

A hands-on learning repository documenting my journey into Agentic AI through implementations, experiments, notes, and projects.

## About This Repository

This repository contains practical implementations and experiments created while learning modern Agentic AI systems.

The learning journey will cover:

- Large Language Models (LLMs)
- LangChain
- Retrieval-Augmented Generation (RAG)
- Advanced RAG techniques
- AI Agents and Tool Calling
- LangGraph
- Agentic RAG
- Multi-Agent Systems
- Guardrails
- Evaluation of AI Systems
- Model Context Protocol (MCP)
- Production-ready Agentic AI applications

## Learning Approach

For every major topic, I aim to:

1. Understand the underlying concepts and architecture.
2. Implement the concepts through code.
3. Experiment with different approaches and configurations.
4. Document important observations and lessons learned.
5. Build mini-projects and real-world Agentic AI applications.

## Repository Structure

The repository structure will evolve as I progress through the course.

Each major module will contain relevant code, experiments, notes, and supporting resources.

## Projects

Projects and experiments will be added throughout the learning journey.

The final goal is to build and deploy a production-oriented Agentic AI system demonstrating concepts such as tool usage, reasoning workflows, retrieval, memory, evaluation, guardrails, and human-in-the-loop interaction.
